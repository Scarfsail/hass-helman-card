How should the selected slot be highlighted in the chart?

All options make the full chart clickable. The difference is how the selected slot looks visually.

A — Filled band

Semi-transparent blue column spans full chart height + impact strip. Clear spatial grouping across both areas.

Pros

  • Visually unambiguous
  • Connects chart to strip

Cons

  • May obscure the data lines slightly

B — Hairline + marker

A single vertical line with a small triangle marker at the top. Minimal obstruction of the chart data.

Pros

  • Doesn't cover data
  • Precise cursor feeling

Cons

  • Less obvious "slot" area
  • Thinner = harder to see

C — Slot-width bordered band

A band exactly as wide as one 15-min slot, with a subtle border — same style as the existing row highlight, but vertical and across the chart.

Pros

  • Consistent with table row style
  • Shows slot boundaries precisely

Cons

  • Very narrow — can be hard to see
